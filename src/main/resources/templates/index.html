LOCAL CONVERTER

文档格式互转工具

在本地电脑上完成 PDF、Word、PPT、Markdown、HTML、TXT 等格式转换。 文件不会上传到外部服务器，核心转换依赖你本机安装的 LibreOffice 与 Pandoc。

支持说明

  • Word / ODT / RTF / TXT / HTML 之间互转
  • PPT / PPTX / ODP 互转及导出 PDF
  • Markdown 转 HTML / DOCX / PDF / TXT
  • PDF 当前支持提取文本到 TXT / HTML / DOCX
